# Expense Tracker 💸

## Overview

Expense Tracker is a Python-based web application built using
**Streamlit** and **Matplotlib** that helps users monitor and analyze
their monthly finances.

The application allows users to enter their monthly income and
categorize expenses into:

-   🛒 Living Essentials
-   ❤️ Health
-   🏠 Home Care
-   🎓 Education
-   🎬 Entertainment
-   ✈️ Travel
-   👕 Clothing

Based on the entered data, the application calculates:

-   Total Expenses
-   Total Savings
-   Savings Percentage
-   Expense Percentage

It also generates a **Financial Health Score** based on savings rate,
spending habits, and overall expense distribution.

## Installation

1.  Clone the repository.

``` bash
git clone <your-repository-url>
```

2.  Navigate to the project folder.

``` bash
cd Expense-Tracker
```

3.  Install the required libraries.

``` bash
pip install -r requirements.txt
```

4.  Run the application.

``` bash
streamlit run app.py
```
